# Yet Another Todo

Очередная реализация todo-списка для погружения в веб-разработку

### Структура репозитория

```text
.
├── client - фронтенд на HTML, CSS, JavaScript
├── client-ts - фронтенд на HTML, CSS, TypeScript 
├── react - фронтенд на React
└── server - бэкенд
```
